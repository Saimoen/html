Texte -- Ecrire avec HTML

Pour écrire un texte basique il faut la balise "p"

Pour un texte en italique, il faut la balise "em"
Et pour un texte en gras, il faut la balise "strong"
L'élément en "span" ne revient pas à la ligne

Photos -- Mettre des photos avec HTML

Pour insérer des images il faut la balise "img src="

On peut changer la largeur grâce à la balise "height" à placer dedans "img"

Listes -- Lister avec HTML

Pour faire des listes il existe 2 balises tels que: "ol" et "ul"

"ol" permet de faire des listes classiques:

  • Maîtriser HTML
  • Apprendre CSS
  • Découvrir Javascript

"ul" permet de faire des listes numérotées:

  1. Maîtriser HTML
  2. Apprendre CSS
  3. Découvrir Javascript

Tableaux -- Faire des tableaux avec HTML

Pour faire des tableaux, il faut utiliser la balise "table"

La balise "caption" permet de mettre un titre au tableau, pour remplir l'en-tête du tableau la balise "thead" est utilisée, le "tbody" permet de remplir le tableau, comme pour les listes "ol" = "tr" et "li" = "td"

Pour créer un formulaire à choix, il faut utiliser la balise "option", utiliser l'"id" du "label" pour que ça corresponde
